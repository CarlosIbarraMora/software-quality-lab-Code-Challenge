## Parte 1 — Módulo `users`

### ¿Qué implementaron?

Desarrollamos la gestión de los usuarios, desde la lógica de negocio
hasta la pruebas.

Implementamos el servicio para los usuarios con la lógica de negocio,
validaciones de datos y manejo de excepciones, utilizando un controller 
advice para manejar las respuestas Http adecuadas(Emiliano).

Además, se realizaron pruebas unitarias enfocadas a validar la lógica del
servicio para los usuarios y la validación de emails(Carlos). Por último, 
se desarrollaron pruebas de integración para revisar el funcionamiento del
controlador, incluyendo endpoints y respuestas(Andrés).

Uno de los mayores retos fue la implementación de las pruebas, debido a que,
como no desarrollamos todo el código y no estábamos familiarizados
al 100%, nos confundía en el proceso de debugging. Pero se logró algo :)
